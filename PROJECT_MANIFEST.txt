SUPPORT & TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════
Common Issues:
  Q: Port 3000 already in use?
  A: Change port in vite.config.js or kill process

  Q: MongoDB connection error?
  A: Ensure MongoDB is running (mongod command)

  Q: CORS error?
  A: Update APP_URL in backend .env

  Q: JWT verification failed?
  A: Check token format and expiration

  Q: Module not found?
  A: Run npm install again in affected directory

For more help, see SETUP.md troubleshooting section.
