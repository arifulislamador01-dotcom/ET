## ❓ TROUBLESHOOTING

**Port already in use?**
```bash
# Change port in .env or kill process
```

**MongoDB not connecting?**
```bash
# Ensure MongoDB is running
mongod
```

**Module errors?**
```bash
npm install
rm -rf node_modules package-lock.json
npm install
```

**CORS error?**
```bash
# Update APP_URL in .env
APP_URL=http://localhost:3000
```
